# Instruções de usos

## 📦 Abrindo o Processador no Logisim
1. Faça o download ou clone do [Enigma.circ](./Enigma.circ)

2. Abre o programa do Logisim

3. Clique em **Arquivo** na aba de ferramentas
   ><img width="560" height="318" alt="Captura de tela 2026-05-05 085343" src="https://github.com/user-attachments/assets/124ae407-7b9d-4068-af3e-dc5f45f6f98d" />
   
4. Clique em **Abrir** na aba suspensa
   ><img width="560" height="318" alt="Captura de tela 2026-05-05 085713" src="https://github.com/user-attachments/assets/d00a6c85-9a56-48d4-be76-10e7da96d5a9" />
   
5. Selecione o Enigma.circ e clique em **Abrir**
   ><img width="560" height="318" alt="Captura de tela 2026-06-02 082647" src="https://github.com/user-attachments/assets/237b334d-c11e-48cb-be3f-2536728b39e1" />

6. Pronto! O Enigma foi aberto no Logisim!
   ><img width="560" height="318" alt="Captura de tela 2026-06-02 082836" src="https://github.com/user-attachments/assets/11a7b8a2-c575-4b76-9345-5bcee2c40294" />

   ><img width="560" height="80" alt="Captura de tela 2026-06-02 083253" src="https://github.com/user-attachments/assets/0d582363-3845-4625-98f8-30e86b2c5735" />

   ><img width="560" height="172" alt="Captura de tela 2026-06-02 083622" src="https://github.com/user-attachments/assets/0012ac1c-ee09-43ff-95cd-2a80b264ebd3" />
